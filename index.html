🌊 Neptuno - JSON API

Servidor estático para exponer documentos y ubicaciones geográficas

✓ Esta API expone dos archivos JSON para consumo desde otras aplicaciones. Ideal para usar con Spring Boot, Node.js, Python, etc.

📄 Documentos

Contiene DNI y RUC de personas y empresas

GET /json/documentos

🗺️ Ubicaciones

Ubicaciones geográficas de Perú (23 registros)

GET /json/ubicaciones

💳 Deudas Externas

Registro de deudas externas asociadas a documentos (DNI y RUC)

GET /json/deudas-externas

💚 Health Check

Verificar que el servidor está funcionando

GET /api/health

📚 Código de Ejemplo

JavaScript / Fetch

fetch('/json/documentos')
  .then(r => r.json())
  .then(data => console.log(data));

Java / Spring Boot

RestTemplate rt = new RestTemplate();
Map data = rt.getForObject("/json/documentos", Map.class);

Python / Requests

import requests
r = requests.get('/json/documentos')
data = r.json()

cURL

curl http://localhost:3000/json/documentos
curl http://localhost:3000/json/ubicaciones
curl http://localhost:3000/json/deudas-externas
